Run server.py script on your PC or router, where the base station mote is attached.
Python (2.7+) with PySerial (2.6+) are required to be installed.

Connect to the web interface using a browser, specify IP address or hostname and port 30000.
For example: http://localhost:30000

The server settings can be tuned by changing values in server.cfg file, and then restarting.
